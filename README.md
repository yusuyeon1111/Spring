# Spring

Spring Framework의 특징

1. POJO 방식의 프레임워크이다.
2. 경량 컨테이너로서 자바 객체를 직접 관리함
3. IoC, DI, AOP 를 지원함
4. 확장성, 생산성이 높음

annotation
1. @Controller → FC 가 일을 시킬 수 있는 POJO임을 나타냄
2. url - 메소드 mapping @RequestMapping(””)
3. @Autonwired를 통해 xml파일에서 생성되어 있는 데이터를 사용함
